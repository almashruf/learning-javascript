Sure! JavaScript arrays are versatile and powerful, allowing you to store and manipulate collections of data. Here are some examples demonstrating various array operations: ### 1. Creating Arrays #### Literal Notation ```javascript let fruits = ["Apple", "Banana", "Mango"]; ``` #### Using the Array Constructor ```javascript let cars = new Array("Tesla", "BMW", "Audi"); ``` ### 2. Accessing Array Elements ```javascript let firstFruit = fruits[0]; // "Apple" let secondCar = cars[1]; // "BMW" ``` ### 3. Modifying Arrays #### Adding Elements ```javascript fruits.push("Orange"); // Adds "Orange" to the end fruits.unshift("Strawberry"); // Adds "Strawberry" to the beginning ``` #### Removing Elements ```javascript fruits.pop(); // Removes the last element ("Orange") fruits.shift(); // Removes the first element ("Strawberry") ``` ### 4. Iterating Over Arrays #### Using a `for` Loop ```javascript for (let i = 0; i < fruits.length; i++) { console.log(fruits[i]); } ``` #### Using `forEach` ```javascript fruits.forEach(function(fruit) { console.log(fruit); }); ``` #### Using `for...of` ```javascript for (let fruit of fruits) { console.log(fruit); } ``` ### 5. Array Methods #### `map()` ```javascript let numbers = [1, 2, 3, 4]; let doubled = numbers.map(function(num) { return num * 2; }); // doubled: [2, 4, 6, 8] ``` #### `filter()` ```javascript let evenNumbers = numbers.filter(function(num) { return num % 2 === 0; }); // evenNumbers: [2, 4] ``` #### `reduce()` ```javascript let sum = numbers.reduce(function(accumulator, currentValue) { return accumulator + currentValue; }, 0); // sum: 10 ``` #### `find()` ```javascript let found = numbers.find(function(num) { return num > 2; }); // found: 3 ``` #### `findIndex()` ```javascript let index = numbers.findIndex(function(num) { return num > 2; }); // index: 2 ``` #### `sort()` ```javascript let sortedFruits = fruits.sort(); // sortedFruits: ["Apple", "Banana", "Mango"] ``` #### `reverse()` ```javascript let reversedFruits = fruits.reverse(); // reversedFruits: ["Mango", "Banana", "Apple"] ``` ### 6. Array Destructuring ```javascript let [first, second] = fruits; // first: "Mango", second: "Banana" ``` ### 7. Spread Operator ```javascript let moreFruits = [...fruits, "Pineapple"]; // moreFruits: ["Mango", "Banana", "Apple", "Pineapple"] ``` ### 8. Array Length ```javascript let length = fruits.length; // 3 ``` ### 9. Multi-dimensional Arrays ```javascript let matrix = [ [1, 2, 3], [4, 5, 6], [7, 8, 9] ]; let firstRow = matrix[0]; // [1, 2, 3] let firstElement = matrix[0][0]; // 1 ``` ### 10. Array Copying #### Using `slice()` ```javascript let copyOfFruits = fruits.slice(); // copyOfFruits: ["Mango", "Banana", "Apple"] ``` #### Using `concat()` ```javascript let anotherCopy = [].concat(fruits); // anotherCopy: ["Mango", "Banana", "Apple"] ``` ### 11. Checking if a Variable is an Array ```javascript Array.isArray(fruits); // true ``` ### 12. Combining Arrays ```javascript let combined = fruits.concat(cars); // combined: ["Mango", "Banana", "Apple", "Tesla", "BMW", "Audi"] ``` ### 13. Array from a String ```javascript let string = "Hello"; let charArray = Array.from(string); // charArray: ["H", "e", "l", "l", "o"] ``` ### 14. Array from Arguments ```javascript function createArray() { return Array.from(arguments); } let arr = createArray(1, 2, 3, 4); // arr: [1, 2, 3, 4] ``` ### 15. `every()` and `some()` #### `every()` ```javascript let allEven = numbers.every(function(num) { return num % 2 === 0; }); // allEven: false ``` #### `some()` ```javascript let someEven = numbers.some(function(num) { return num % 2 === 0; }); // someEven: true ``` These examples cover a wide range of array operations in JavaScript, demonstrating their flexibility and utility in managing and manipulating collections of data.
